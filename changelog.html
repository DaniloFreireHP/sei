  • BUG FIX: Corrigido ponto de controle cores.

Versão 2.2 - Inclui funcionalidades específicas para a Anatel

  • (Anatel) Foi adiciona ferramenta para preencher automaticamente o andamento do processo quando for enviar oficio ao protocolo.
  • (Anatel) Foi incluido a opção do ponto de controle em cores.
  • (Anatel) Foi incluido a ferramenta "Click menos", para autopreecher os dados de novos documentos externo.
  • BUG FIX: Corrigido a checagem de blocos de assinatura.

Versão 2.1

  • Foi adiciona ferramenta para colocar o menu do sistema suspenso, acessível no logo do SEI.
  • Habilitado a opção para voltar à ordenação original das tabelas de processos.
  • Salva a ordenação das tabelas de processos.
  • Ajustes no tema black.

Versão 2.0

  • Foi incluido o tema preto (SEI Black Theme).
  • Foi incluido a funcionalidade "Checar bloco de assinaturas".

Versão 1.6.3

  • Foi incluido o indentificador "++" no logo do SEI.
  • Foi incluido a mensagem de confirmação ao salvar as configurações.
  • Corrigir a duplicação das colunas ao atualizar.
  • Ao instalar ou atualizar abre a página do complemento (incluir em opções).
  • Deve ser utilizado o stylish "SEI Black Theme 1.5" para usar o tema preto (Atulizar para funcionar corretamente). SEI Black Theme 1.5 - Segir conforme o Manual para autalizar.
